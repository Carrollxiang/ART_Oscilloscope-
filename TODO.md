## P1 — 扫频锁频核心功能

- [ ] **协议升级: 时间戳/序号**: STM32 每条数据附带采样序号 (`n=12345`) 或时间戳 (`t=12345`)，Python 直接用它构造时间轴，消除均匀采样假设带来的 jitter。
- [ ] **扫频参数配置**: UI 支持设置扫频起始/终止频率、扫频速度、RF 功率等参数，下发给硬件。
- [ ] **V(f) 响应函数**: 扫频结束后，根据时间轴 + 频率扫描参数，将 V(t) 映射为 V(f)，用于线型拟合。
- [ ] **线型拟合**: 对 V(f) 进行 Lorentzian / Gaussian 拟合，输出中心频率 `f0`、线宽 `Γ`、拟合优度 `R²`。
- [ ] **扫频时序同步**: TTL + RF 同时启停，Python 端据此确定扫频窗口起点和终点。
